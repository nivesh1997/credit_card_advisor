This is an AI based credit card advisor which is built using Fast API, Langchain and Google Gemini LLM model. It works on salary, spending Limit , reward type, annual fee of card, and other perks also.

Each credit card is scored on:
Income eligibility
Credit score
Spending categories (fuel, dining, travel, shopping, groceries)
Rewards type match
Perks like lounge access
Estimated annual rewards
